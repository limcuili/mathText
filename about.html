Pages

I want each module listed in course outline to link to a page containing the module details, and a link to the module notes. Maybe a options menu when you hover over the module title, which lets you select 'module description' or 'module notes'

Cloud Firestore

I will use Cloud Firestore to save the text for module details, including lecturer names and module description. I think Cloud Firestore's database is good for this because it is not a dynamic page, but rather a static one. In fact, the whole website is going to be pretty static. I am choosing this database becuase it is NoSQL and I am a little familiar with it.

The actual math

1. I will need a database where I can store MathJax's syntax, which will be able to be pushed onto the html pages. These will then be read by MathJax and displayed on the html page.
